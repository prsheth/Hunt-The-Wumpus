Hunt-The-Wumpus

Introduction
It is a simple game based on a hide and seek format featuring a mysterious monster that sits inside a deep network of rooms.
This is a text based game on console is built using C#.

Gameplay
The objects includes the Wumpus, Super bats -2 , Pits-2. There are two possible actions one is move and shoot.The player can move around the rooms by selecting the number of the room.
However the player has to be careful and attentive for the warning messages which act like clues to hunt the wumpus.On entering rooms that contain the pits or the bats the player looses a life.
